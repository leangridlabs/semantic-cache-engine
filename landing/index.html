How it works

Three steps. No config. Works with any codebase.

🔍

Ingest

The engine scans your repo, chunks the source into semantic units, and builds a local reasoning graph. Runs once per repo — incrementally updates on file changes.

💾

Cache

Every question Copilot answers about your codebase creates a reasoning card — stored in a local SQLite database on your machine. No cloud, no telemetry, no source code sent anywhere.

⚡

Recall

Next session, the same questions hit the cache instantly. No LLM call, no token spend. Share an encrypted bundle with your team and they start warm too.
